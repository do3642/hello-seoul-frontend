#  안녕, 서울

## 📚 프로젝트 개요

이 프로젝트는 **서울 관광지 소개 사이트**를 개발하는 프로젝트입니다.  
사용자들에게 서울의 주요 관광지, 축제를 소개하고, 관광지에 대한 정보를 각 구별, 검색을 통해 제공합니다.  
프로젝트는 **React**와 **Spring Boot**를 사용하여 프론트엔드와 백엔드를 구현합니다.

## ⏳ 작업 기간 및 내용

- **작업 기간**: 2024년 12월 31일 ~ 2025년 1월 22일
- **내용**:
  - 서울의 주요 관광지 정보 제공
  - 계절별 맞춤 관광지 추천 기능 구현
  - 지하철역에 따른 관광지 추천 제안
  - 진행중인 축제 관련 정보 제공

## 💡 기능 소개

- **서울 관광지 목록**: 서울의 주요 관광지 리스트를 확인할 수 있습니다.
- **관광지 상세 정보**: 각 관광지에 대한 상세 정보를 제공합니다.
- **검색 기능**: 특정 관광지를 검색할 수 있습니다.
- **추천 시스템**: 계절별 관광지를 추천해주는 기능을 제공합니다.

## 🔧 기술 스택

- **프론트엔드**: React, Axios
- **백엔드**: Spring Boot, MySQL

## 👨‍💻 팀 구성

이 프로젝트는 총 4명이 참여한 팀 프로젝트입니다:
- **이영찬** (팀 리더)
- **최하영**
- **권민수**
- **허경**

## 🚀 설치 및 실행 방법

### 1. **프론트엔드 실행**

- 프로젝트의 프론트엔드 폴더로 이동합니다.

- **프로젝트 클론**
  ```bash
  git clone https://github.com/do3642/hello-seoul-frontend.git
  ```

- 클론 후 develop 브랜치 확인

- **필요한 패키지 설치**
  프로젝트 디렉토리 내에서 아래 명령어로 의존성 패키지를 설치합니다.
  ```bash
  npm install
  ```

- **.env.local 파일 생성**
  프로젝트의 루트 디렉토리에 `.env.local` 파일을 생성하고, 아래와 같이 서버 URL을 설정합니다.
  ```bash
  VITE_REACT_APP_SERVER_URL=http://localhost:8888
  ```

- **프로젝트 실행**
  개발 서버를 실행하여 애플리케이션을 실행합니다.
  ```bash
  npm run dev
  ```

기본적으로 `http://localhost:5173`에서 프론트엔드가 실행됩니다.

> 📌 **참고**: 백엔드 서버가 별도로 실행되어야 하며, 프론트엔드와 백엔드 서버가 서로 통신합니다.
